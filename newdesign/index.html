What is Jamulus?

The Jamulus software enables musicians to perform real-time jam sessions over the internet.

How does Jamulus work?

There is one server running the Jamulus server software which collects the audio data from each Jamulus client, mixes the audio data and sends the mix back to each client.

Jamulus is Open Source

Jamulus is Open Source software (GPL, GNU General Public License) and runs under Windows (ASIO), MacOS (Core Audio) and Linux (Jack). It is based on the Qt framework and uses the OPUS audio codec.

Contribute

Ideas? Found a bug? Want to contribute? Since Jamulus is open source you can help us! Find the project on SourceForge and the source code on GitHub!

What do I need?

Internet: The required minimum internet connection speed is 200 kbps for the up- and downstream. The ping time (i.e. round trip delay) from your computer to the server should not exceed 40 ms average. To get best results you should not use a wireless internet connection (WIFI). This might leed to more latency and bad audio quality. We recommend to use LAN.

Audio interface: Using an audio interface is recommend for high quality sound and low latency.
